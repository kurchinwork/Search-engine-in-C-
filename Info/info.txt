Запуск консольного приложения и подготовка к работе.

    1. В конфигурационном файле перед запуском приложения задаются названия
файлов, по которым движок будет осуществлять поиск.

    Выполнить следующее:
    Файл конфигурации config.json

    {
        "config": {
            "name": "SkillboxSearchEngine",
            "version": "0.1",
            "max_responses": 5
    },
        "files": [
            "../resources/file001.txt",
            "../resources/file002.txt",
            "../resources/file003.txt",
            "../resources/file004.txt",
        ]
    }

    В поле "name" - обязательно установить имя конфигурации;
    В списке "files" - перечислить все файлы по которым будет
   осуществляться поиск, данные файлы обязательно
   разместить в папке "resources";
